# SkillMatch2050  
Protótipo desenvolvido para a Global Solution – tema: *Novas Carreiras, Novas Tecnologias, Infinitas Possibilidades*.

O objetivo é recomendar carreiras do futuro para diferentes perfis, utilizando Python e estruturas de pensamento computacional.

---

## Tecnologias utilizadas
- Python
- Pandas (DataFrame)
- Lógica de decisão
- Funções e subfunções
- Estruturas de repetição, entrada e saída

---

## O que o código faz?
O sistema coleta perfis de usuários (nome, skills, interesses, disponibilidade para aprender) e recomenda **3 carreiras do futuro** com base em compatibilidade, impacto e afinidade tecnológica.

Ele gera:
- Relatório na tela
- DataFrame organizado
- Arquivo CSV opcional
